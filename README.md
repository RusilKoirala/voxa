# Voxa 

Voxa a social media platform like Reddit :). Here you can make communities like /c and invite people to your communities. People can post their opinion on the community! On the post people and comment and other people can reply over his comment. The post, comments, reply all get ranked by upvote and downvote. The user can visit other communites through search and view others post (yee like reddit).

## What this project does 

- User registration and login with JWT authentication.
- Create and. join communitiess (like subreddits)
- Post CONTENT to communities with title, descriptions!
- Comment on posts with nested reply threadss.
- Upvote and downvote posts and comments.
- Browse user profiles with acitvity stats.
- Search functionality

## How the app flows

1. User registers and logs in.
2. Users browse existing communites or create their own
3. Within a community the users can posts and reply to comments
4. Other users can comment on post and reply to comments (Yes like reddit)
5. All content is ranked by upvotes and downvotes (YES AIIII or i will say algorithms are used to rank the content)
6. Users have profile showing their activity and stats

## GETTING STARTEDD

### Backend Setup

1. Navigate to backend/ directory
2. Install dependencies: pnpm install
3. Set up environment variables (copy .env.example to .env )
4. Run database migrations: pnpm db:migrate
5. Start development server: pnpm dev

### Frontend Setup

1. Navigate to frontend/ directory
2. Install dependencies: pnpm install
3. Set up environment variables
4. Start development server: pnpm dev
